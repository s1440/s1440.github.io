Welcome to My CIT 281 Portfolio

This portfolio showcases my work for the CIT 281 course at the University of Oregon. It includes individual lab exercises and major projects that demonstrate my understanding of JavaScript, Node.js, GitHub, APIs, servers, and more. Each lab and project has its own page with descriptions, code, and live previews.

The goal of this portfolio is not only to track my progress but also to demonstrate my learning outcomes through real code and development tools. It is hosted using GitHub Pages and built using basic HTML and CSS.

Thank you for visiting!
